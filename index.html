關於我們

滙漢科技有限公司（Huihan Inc.）為 匯彩控股有限公司 （香港上市公司，股份代號：1180）旗下全資子公司，總部設立於台中。 匯彩控股長年深耕亞洲娛樂與科技產業，於香港聯合交易所主板掛牌上市，財務穩健，企業治理透明，為集團在全球數位娛樂領域的策略性布局提供強力後盾。

滙漢科技有限專注於線上遊戲的開發與創新，致力打造多樣化、跨平台的高品質遊戲內容。 我們的開發團隊由來工程師與創意人才組成，涵蓋捕魚遊戲、Table Games、Poker Games、Lottery、Live Casino、老虎機遊戲等多種類型， 並也想試著開發休閒類的遊戲，積極拓展遊戲類型的可能性，勇於挑戰創新與技術整合的極限。

在技術層面，我們採用業界主流且高效的技術架構：

  • 前端開發：使用 Cocos Creator + TypeScript 為主力開發工具，支援快速跨平台部署，兼具效能與靈活性。
  • 後端開發：以 Go 語言 結合 MySQL 資料庫，打造高效穩定的遊戲伺服器與服務端邏輯。 也採用微服務 + Docker 的功能，帶來了高度的彈性、擴展性、可維護性與部署效率。
  • 資料處理與分析：運用 Python 處理數據、撰寫工具與支援 AI 邏輯開發，提升決策效率與產品精準度。

我們重視開發流程中的溝通與協作，致力於建立開放、尊重、持續學習的團隊文化， 吸引對遊戲充滿熱情、技術導向的夥伴加入。無論是資深開發者還是新銳創作者， 滙漢都是一個可以實現夢想、磨練技能、共同成長的舞台。

我們的願景是：「從台中出發，打造世界級的線上娛樂體驗。」
